# 店舗総合管理スプレッドシート｜業務自動化サンプル

現役バー経営者が **実店舗で実際に運用している**  
勤怠管理・給与計算・売上管理を自動化する  
スプレッドシート／業務改善仕組みの紹介リポジトリです。

手書き・手計算で発生していた  
「時間ロス」「計算ミス」「確認作業」を極限まで削減することを目的に作成しました。

---

## 🔧 この仕組みでできること

- 勤怠管理（出勤・退勤入力 → 勤務時間自動計算）
- 日跨ぎ（深夜勤務）対応
- スタッフ別給与計算（時給・バック計算）
- 経費管理（酒代・家賃・消耗品など）
- レジ金管理（前日差異の自動チェック）
- 売上・顧客管理（担当者・支払方法ごとに自動集計）
- 利益の自動算出
- スタッフ売上ランキングの可視化

※ すべて **Googleスプレッドシート上で完結**します。

---

## 🧠 特徴・強み

- 実際のバー経営で運用中
- 現場目線で「本当に使う機能」だけを実装
- 小規模店舗向け（大げさなPOS・SaaS不要）
- カスタマイズ前提で設計可能
- Excel / スプレッドシートが苦手な方でも運用しやすい構成
